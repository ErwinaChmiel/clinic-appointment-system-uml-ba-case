# Glossary

| Term | Definition |
|---|---|
| Patient | A person using the system to book an appointment. |
| Doctor | A specialist who provides appointment availability. |
| Slot | A time interval available for a specific doctor and service. |
| Appointment | A confirmed or temporary reservation of a slot by a patient. |
| Prepayment | Partial or full payment required before appointment confirmation. |
| Payment Gateway | External system responsible for payment authorization. |
| Notification | SMS/e-mail message sent to the patient. |
