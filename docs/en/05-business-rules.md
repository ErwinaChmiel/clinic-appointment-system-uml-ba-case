# Business rules

| ID | Rule |
|---|---|
| R-01 | One slot can be assigned to only one confirmed appointment. |
| R-02 | A temporary slot reservation expires after a defined time. |
| R-03 | Prepayment is required only for services marked as requiring prepayment. |
| R-04 | An appointment can be confirmed only after the payment condition is met. |
| R-05 | After failed payment, the slot must be released. |
| R-06 | Appointment confirmation must include date, time, doctor and service name. |
| R-07 | Appointment cancellation may be time-restricted according to clinic policy. |
