# Business Rules

| ID | Business rule |
|---|---|
| BR-01 | An appointment slot can be assigned to only one active appointment. |
| BR-02 | A temporary slot reservation expires after 10 minutes. |
| BR-03 | An appointment can be confirmed only after successful payment authorization. |
| BR-04 | A patient can cancel an appointment no later than 24 hours before the appointment time, unless the service has different rules. |
| BR-05 | The system must send a confirmation after every successful booking and payment. |
| BR-06 | A failed payment releases the temporarily reserved appointment slot. |
| BR-07 | Each booking requires online prepayment before appointment confirmation. |
| BR-08 | Each appointment must have a clear and unambiguous status. |
