# Business rules

| ID | Rule |
|---|---|
| BR-01 | A slot can be assigned to only one active appointment. |
| BR-02 | A temporary slot reservation expires after 10 minutes. |
| BR-03 | An appointment requiring prepayment can be confirmed only after successful payment authorization. |
| BR-04 | A patient can cancel an appointment no later than 24 hours before the slot unless a service has different rules. |
| BR-05 | The system must send a confirmation after every successful booking. |
| BR-06 | Failed payment releases the temporarily reserved slot. |
