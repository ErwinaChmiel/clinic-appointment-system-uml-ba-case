# Słownik pojęć

| Termin | Definicja |
|---|---|
| Pacjent | Osoba rezerwująca wizytę medyczną. |
| Lekarz | Osoba wykonująca usługę medyczną. |
| Termin | Konkretny przedział czasu dostępny dla wizyty. |
| Tymczasowa rezerwacja | Krótkotrwała blokada terminu podczas procesu rezerwacji, utrzymywana do czasu zakończenia płatności lub wygaśnięcia sesji. |
| Przedpłata | Płatność online wymagana przed potwierdzeniem wizyty. |
| Płatność | Proces autoryzacji przedpłaty online przez zewnętrzną bramkę płatności. |
| Nieudana płatność | Płatność, która została odrzucona, przerwana lub nie została zakończona w wymaganym czasie. |
| Wizyta potwierdzona | Wizyta, która została potwierdzona po pozytywnej autoryzacji płatności i jest widoczna w harmonogramie. |
| Powiadomienie | Wiadomość e-mail lub SMS wysyłana do pacjenta w związku z rezerwacją, zmianą lub anulowaniem wizyty. |
| Harmonogram | Zestaw dostępnych terminów lekarza zarządzany przez recepcję lub administratora. |
| Status wizyty | Aktualny stan wizyty, np. oczekuje na płatność, potwierdzona, płatność nieudana, anulowana, nieodbyta, w trakcie lub zakończona. |
| Bramka płatności | Zewnętrzny system obsługujący autoryzację płatności online. |
| System powiadomień | Komponent lub usługa odpowiedzialna za wysyłanie wiadomości e-mail lub SMS do pacjentów. |
