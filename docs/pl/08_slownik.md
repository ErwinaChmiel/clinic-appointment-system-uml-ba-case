# Słownik pojęć

| Termin | Definicja |
|---|---|
| Pacjent | Osoba rezerwująca wizytę medyczną. |
| Lekarz | Osoba wykonująca usługę medyczną. |
| Termin | Konkretny przedział czasu dostępny dla wizyty. |
| Tymczasowa rezerwacja | Krótkotrwała blokada terminu podczas procesu rezerwacji. |
| Przedpłata | Płatność wymagana przed potwierdzeniem wizyty. |
| Wizyta potwierdzona | Wizyta spełniająca warunki rezerwacji i widoczna w harmonogramie. |
| Płatność |Proces autoryzacji przedpłaty online przez zewnętrzną bramkę płatności. |
| Powiadomienie | Wiadomość e-mail lub SMS wysyłana do pacjenta w związku z rezerwacją, zmianą lub anulowaniem wizyty. |
| Harmonogram | Zestaw dostępnych terminów specjalisty zarządzany przez recepcję lub administratora.
| Status wizyty | Aktualny stan wizyty, np. oczekuje na płatność, potwierdzona, anulowana lub zakończona. |
