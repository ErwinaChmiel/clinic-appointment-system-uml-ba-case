# Zakres i interesariusze

## Zakres MVP

Wersja MVP obejmuje:

1. wyszukiwanie specjalisty,
2. wybór dostępnego terminu,
3. tymczasową rezerwację terminu,
4. obsługę przedpłaty, jeśli jest wymagana,
5. potwierdzenie wizyty,
6. wysłanie wiadomości potwierdzającej e-mail lub SMS,
7. anulowanie wizyty przez pacjenta lub personel,
8. podstawowe zarządzanie harmonogramem.

## Interesariusze

| Interesariusz | Potrzeba biznesowa |
|---|---|
| Pacjent | Szybka i czytelna rezerwacja wizyty online. |
| Lekarz / specjalista | Aktualny harmonogram i mniej zmian ręcznych. |
| Recepcja | Mniej połączeń i mniej ręcznego zarządzania terminami. |
| Administrator placówki | Kontrola usług, specjalistów, harmonogramów i reguł płatności. |
| Dostawca płatności | Bezpieczna autoryzacja płatności. |
| System powiadomień | Dostarczenie potwierdzeń i komunikatów do pacjentów. |

## Poza zakresem

- dokumentacja medyczna,
- recepty elektroniczne,
- integracje z systemami publicznymi,
- pełne rozliczenia księgowe.
