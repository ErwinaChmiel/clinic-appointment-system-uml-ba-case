# Słownik pojęć

| Pojęcie | Definicja |
|---|---|
| Pacjent | Osoba korzystająca z systemu w celu rezerwacji wizyty. |
| Lekarz | Specjalista udostępniający terminy wizyt. |
| Termin | Przedział czasowy dostępny dla konkretnego lekarza i usługi. |
| Wizyta | Potwierdzona lub tymczasowa rezerwacja terminu przez pacjenta. |
| Przedpłata | Częściowa lub pełna płatność wymagana przed potwierdzeniem wizyty. |
| Bramka płatności | Zewnętrzny system obsługujący autoryzację płatności. |
| Powiadomienie | Komunikat SMS/e-mail wysyłany do pacjenta. |
