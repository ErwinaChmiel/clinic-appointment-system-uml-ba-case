# Reguły biznesowe

| ID | Reguła |
|---|---|
| R-01 | Jeden termin może być przypisany tylko do jednej potwierdzonej wizyty. |
| R-02 | Tymczasowa rezerwacja terminu wygasa po określonym czasie. |
| R-03 | Przedpłata jest wymagana tylko dla usług oznaczonych jako wymagające przedpłaty. |
| R-04 | Wizyta może zostać potwierdzona dopiero po spełnieniu wymogu płatności. |
| R-05 | Po nieudanej płatności termin musi zostać zwolniony. |
| R-06 | Potwierdzenie wizyty musi zawierać datę, godzinę, lekarza i nazwę usługi. |
| R-07 | Anulowanie wizyty może być ograniczone czasowo zgodnie z polityką placówki. |
