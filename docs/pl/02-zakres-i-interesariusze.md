# Zakres i interesariusze

## Zakres rozwiązania

System obejmuje wyszukiwanie dostępnych terminów, rezerwację wizyty, opcjonalną przedpłatę, anulowanie wizyty, zarządzanie dostępnością lekarzy oraz wysyłkę powiadomień.

## Poza zakresem

- dokumentacja medyczna pacjenta,
- teleporady,
- wystawianie recept,
- integracja z systemami publicznymi,
- zaawansowana analityka finansowa.

## Interesariusze

| Interesariusz | Odpowiedzialność | Potrzeba |
|---|---|---|
| Pacjent | Rezerwuje i anuluje wizyty | Szybki dostęp do terminów |
| Recepcja | Obsługuje wyjątki i grafik | Mniej ręcznej pracy |
| Lekarz | Udostępnia terminy | Aktualny grafik |
| Administrator | Konfiguruje system | Kontrola użytkowników i usług |
| Zarząd placówki | Ocena efektywności | Mierzalne usprawnienie procesu |
