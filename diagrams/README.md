# Diagramy UML / UML diagrams

## PL

Diagramy są dostępne w dwóch formatach:

- `svg/pl` — podgląd w GitHub,
- `drawio/pl` — edytowalne źródła diagrams.net.

## EN

Diagrams are available in two formats:

- `svg/en` — GitHub preview,
- `drawio/en` — editable diagrams.net sources.

## Zasada spójności / Consistency rule

Każdy diagram ma wersję polską i angielską z tym samym układem. Różni się wyłącznie tekst etykiet.

Each diagram has a Polish and English version with the same layout. Only labels differ.
