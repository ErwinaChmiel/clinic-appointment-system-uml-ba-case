# Diagrams

This folder contains only the diagram assets used by the main repository README.

## Structure

```text
diagrams/
├── png/
│   ├── pl/   # Polish diagram previews
│   └── en/   # English diagram previews
└── drawio/
    ├── pl/   # Polish diagrams for diagrams.net / draw.io
    └── en/   # English diagrams for diagrams.net / draw.io
```

## Formats

- `png` files are used as GitHub-friendly previews.
- `drawio` files can be opened in diagrams.net / draw.io.
