# Diagrams

This folder contains UML diagrams in two language versions.

## Structure

```text
diagrams/
├── drawio/
│   ├── pl/   # editable Polish diagrams
│   └── en/   # editable English diagrams
└── svg/
    ├── pl/   # Polish SVG previews
    └── en/   # English SVG previews
```

## Editing

Open `.drawio` files in diagrams.net / draw.io. SVG files are used for GitHub preview in the main README.

## Quality rules used in this version

- identical layout between PL and EN versions,
- orthogonal connectors,
- no connectors routed through shapes,
- no line crossings in the activity diagram,
- consistent colors and typography,
- separate editable and preview formats.
