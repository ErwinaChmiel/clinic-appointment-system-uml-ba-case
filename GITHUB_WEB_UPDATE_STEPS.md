# GitHub web update steps

## Option A — replace the whole repository content

1. Download this ZIP and extract it locally.
2. Open your GitHub repository in the browser.
3. Delete old folders/files that should be replaced, especially `README.md`, `docs`, `diagrams`, `backlog`, `templates`.
4. Upload the extracted project content using `Add file → Upload files`.
5. Commit with the message:

```text
Replace project with bilingual polished UML version
```

## Option B — safer update if you do not want to delete everything

Upload and overwrite these paths:

```text
README.md
docs/
diagrams/
backlog/
templates/
```

Commit message:

```text
Add bilingual professional UML diagrams and documentation
```
