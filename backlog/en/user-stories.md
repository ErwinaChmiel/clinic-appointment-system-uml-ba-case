# Backlog — user stories

| ID | User story | Acceptance criteria |
|---|---|---|
| US-01 | As a patient, I want to search available slots so that I can choose a convenient appointment. | The list shows doctor, service, date and time. |
| US-02 | As a patient, I want to book a selected slot so that my appointment is confirmed. | The slot is temporarily held and then confirmed after the conditions are met. |
| US-03 | As a patient, I want to pay prepayment online so that I can complete booking for a paid service. | After successful payment, the appointment status becomes confirmed. |
| US-04 | As a receptionist, I want to manage appointments so that I can handle exceptions and cancellations. | Reception staff can search, update or cancel an appointment according to permissions. |
| US-05 | As a doctor, I want to manage availability so that patients see accurate slots. | Availability changes affect the list of available slots. |
